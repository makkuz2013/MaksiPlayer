# Features:
  - MP3, WAV, WMA, MIDI, OGG, FLAC, AAC, AIFF playing (it depends on the codecs you've installed)
  - Playlist with saving
  - Adding files to playlist

# Problems
  - Currently, songs with Russian characters in their titles can be added, but they will not persist in the playlist after restarting the application. Contributions and fixes are welcome.
